📊 AI Data Analyst Agent

An interactive Streamlit-based web application that allows users to upload datasets and perform automated data analysis, visualization, and machine learning in a few clicks.

🚀 Features
📁 Upload CSV datasets dynamically
📊 Automatic KPI generation (Rows, Columns, Missing Values)
📈 Data visualization (Histogram, Bar Chart, Box Plot)
🧠 Automated insights (rule-based analysis)
🤖 Machine Learning model training
🎯 Model accuracy display (R² score)

🛠️ Tech Stack
Frontend/UI: Streamlit
Backend: Python
Libraries:
pandas
numpy
matplotlib
seaborn
scikit-learn

⚙️ How to Run Locally
Clone the repository:
git clone https://github.com/Monuwithgithub/AI-Powered-Data-Analyst-Agent.git
Navigate to project folder:
cd AI-Powered-Data-Analyst-Agent
Install dependencies:
pip install -r requirements.txt
Run the app:
streamlit run App.py
